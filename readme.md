Health Insurance Premium Prediction 🏥
This is a Machine Learning project designed to predict health insurance premium costs based on individual parameters such as age, BMI, smoking habits, and more.

📌 Project Overview
The goal of this project is to build a predictive model that helps estimate insurance expenses accurately. Using a Random Forest Regressor, the model analyzes various factors to determine how lifestyle and demographic data impact healthcare costs.

🛠️ Tech Stack
Language: Python

Libraries: Pandas, NumPy, Scikit-learn

Visualization: Seaborn, Matplotlib, Plotly Express

Model: Random Forest Regressor

Environment: Google Colab / VS Code

📊 Key Features & Analysis
The project includes detailed Exploratory Data Analysis (EDA):

Smoker Impact: Analyzes the significant gap in charges between smokers and non-smokers.

BMI Analysis: Visualizes the correlation between Body Mass Index and insurance premiums.

Regional Distribution: Examines how policyholders are spread across different geographic regions.

Predictive Modeling: Uses trained data to predict premium amounts for test cases.
